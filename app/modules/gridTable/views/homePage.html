Create Table Grid with inline edit & save

  1. Create a tabular layout with 3 x 3 grid
  2. Add two links “Add Description” & “Add Title” to each of the grid
  3. On clicking “Add Description” link display a form with Textarea and two buttons to save and cancel edits
  4. When the Textarea has some text and if save button is clicked do the following:
    1. Hide the form with Textarea and two buttons
    2. Display “Add Description” and replace link text with Textarea text.
  5. On clicking “Add Title” link display a form with Input and two buttons to save and cancel edits.
  6. When the Input has some text and if save button is clicked do the following:
    1. Hide the form with Textarea and two buttons
    2. Display “Add Description” and replace link text with Textarea text.
  7. Display error message if the form is submitted with empty fields.
  8. Clicking on “Clear All” button will clear all the edits and replace links with its original text.
